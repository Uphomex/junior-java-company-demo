# 教学文档总览：怎么读这份资料

本目录是配合 `junior-java-company-demo` 项目的"手把手教学包"，目标读者是：

- 学过一点 Java 但没真正写过项目的同学
- 在校或刚毕业、即将进入后端 / 全栈岗位的同学
- 工作 0-1 年，想补全"前后端分离"完整链路的同学

阅读完这份资料后，你应该可以：

1. 看懂一个大厂里典型的 Spring Boot + Vue3 全栈项目结构。
2. 自己照着步骤把项目跑起来。
3. 知道每一段代码用到的语法/框架是什么、为什么这么写。
4. 学会用 Git + GitHub 把代码"云端备份 + 本地 E 盘备份"。

## 用到的关键技术清单

后端：

- **Java 11**：长期支持版本，国内公司常用
- **Spring Boot 2.7**：约定大于配置，极大简化后端工程
- **Spring MVC**：基于注解的 HTTP 接口框架
- **Bean Validation**（`@Valid`、`@NotBlank` 等）：参数校验
- **MyBatis**：手写 SQL 风格的持久层框架
- **H2 + MySQL**：H2 内嵌数据库免安装试跑；MySQL 真实数据库切换
- **Maven**：依赖与构建管理

前端：

- **Vue 3**：渐进式前端框架，国内最流行的前端框架之一
- **TypeScript**：给 JavaScript 加上类型，代码更稳
- **Vite**：极快的前端开发与构建工具
- **Element Plus**：阿里出品的 UI 组件库，企业后台必备
- **Pinia**：Vue 官方推荐的状态管理
- **Vue Router 4**：单页应用前端路由
- **Axios**：浏览器/Node 通用 HTTP 客户端

工具：

- **Git**：版本控制
- **GitHub**：代码托管 + 备份
- **IntelliJ IDEA / VS Code**：IDE
